                              Running a Forecast #

                              There are three sets of instructions for running a forecast, depending on your familiarity with the pipeline and the data you’re starting with.

                              Mock data (your first time forecasting)

                              • If you want to perform your first forecast, you can start here. It provides simple mock data and teaches you how to fit a model for forecasting, how to forecasting, and how to interpret the results.

                              → Mock data walkthrough

                              Climate futures data

                              • If you have your own model fitting data but not your own future scenario data, you can get some here. It details how to forecast using simple CCRP climate futures covariate data.

                              → Climate futures data tutorial

                              Your own data

                              • If you have your own model fitting data and your own future scenario data, you can learn how to forecast here. This details the configuration options and outputs you should know about.

                              → Your own data guide
