# Existing analysis config files as templates

If your data type is common within the NPS Inventory and Monitoring division, it may make sense to create a copy of the appropriate analysis config file template and modify it according to your needs. You can view and edit the analysis configuration files in RStudio, Visual Studio Code, or other code editors. See [Config files]({{< ref "/docs/1-guide/b-config-files/_index.md" >}}) for a list and description of the templates.
